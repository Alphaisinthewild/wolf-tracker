# Wolf Tracker

Local-first health and fitness tracker built with React, Vite, Zustand, IndexedDB, and Recharts.

## Current scope
- Today dashboard
- Daily log
- Progress summary
- 7-day trends
- Goal settings
- Local persistence in IndexedDB

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes
This is the clean v2 rebuild focused on daily usability first, integrations later.
