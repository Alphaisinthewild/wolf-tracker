# Wolf Tracker

Local-first health and fitness tracker built with React, Vite, Zustand, IndexedDB, and Recharts.

## Current scope
- First-run onboarding/setup flow
- Today dashboard
- Daily log
- Body measurements and progress photos
- Progress summary
- 7-day and 30-day trends
- Goal settings
- Local JSON export/import backup
- Local persistence in IndexedDB

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes
This is the clean v2 rebuild focused on daily usability first, integrations later.
